# Title Capture Pro

現在のタブやリンクの情報を、右クリックメニューから素早くキャプチャ・コピーできるChrome拡張機能です。Manifest V3、Vite、TypeScriptで構築されたモダンな設計になっています。

## 主な機能

- **コンテキストメニュー対応**: ページの何もない場所やリンクの上で右クリックして、即座にキャプチャ・モーダルを表示できます。
- **インページ・モーダル**: ページを離れることなく、スタイリッシュなUIでコピー形式を選択できます。
- **多彩なフォーマット**: HTML Link、Markdown、Title Only、URL Only の4つの形式から選択可能です。
- **モダンなUI**: Glassmorphism（グラスモーフィズム）を採用した、Apple風の洗練されたデザインです。
- **ワンクリック・コピー**: 整理された情報を素早くクリップボードにコピーできます。

## 使い方

### 1. 右クリックメニューから起動
Webページ上のリンクや、ページ自体の右クリックメニューから `Title Capture...` を選択します。

### 2. フォーマットの選択
ページ中央に表示されるモーダルから、希望のコピー形式をクリックします。
- **HTML Link**: `<a href="...">Title</a>`
- **Markdown**: `[Title](URL)`
- **Title Only**: ページのタイトルのみ
- **URL Only**: ページのURL（またはリンクのURL）のみ

### 3. コピー完了
クリックすると自動的にクリップボードにコピーされ、画面下部に通知が表示されます。モーダルの外側をクリックするとキャンセルできます。

## 開発者向け情報

### 事前準備
- Node.js (v18 以上)
- npm

### セットアップ
```bash
npm install
```

### ビルド
```bash
npm run build
```
ビルドされたファイルは `dist/` ディレクトリに出力されます。

### Chromeへの読み込み
1. Chromeで `chrome://extensions` を開きます。
2. 右上の **デベロッパー モード** をオンにします。
3. **パッケージ化されていない拡張機能を読み込む** をクリックし、プロジェクト内の `dist/` ディレクトリを選択します。

## 構成
- **src/background.ts**: コンテキストメニューの登録とイベント処理を担当。
- **src/content.ts**: ページ内へのUI（モーダル）の注入とキャプチャ処理、コピー処理を担当。
- **src/style.css**: UIのスタイリング（一部は content.ts 内で定義）。
- **public/manifest.json**: 拡張機能の設定ファイル。
