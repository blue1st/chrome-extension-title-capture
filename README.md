# Title Capture Pro

現在のタブやリンクの情報を、自由に設定したフォーマットで素早くキャプチャ・コピーできるChrome拡張機能です。Manifest V3、Vite、TypeScriptで構築されたモダンな設計になっています。

## 主な機能

- **カスタムフォーマット**: `${title}`、`${url}`、`${favIconUrl}` を使って、独自の出力パターンを定義できます。
- **コンテキストメニュー対応**: ページの何もない場所やリンクの上で右クリックして、即座に情報を取得できます。
- **クラウド同期**: 設定はChromeプロファイルを通じて複数のデバイス間で同期されます。
- **モダンなUI**: Glassmorphism（グラスモーフィズム）を採用したスタイリッシュなデザインで、ダークモードにも完全対応しています。
- **ワンクリック・コピー**: 整理された情報を素早くクリップボードにコピーできます。

## 使い方

### 1. ポップアップからコピー
拡張機能のアイコンをクリックすると、現在のタブの情報が指定したフォーマットで表示されます。`Copy` ボタンを押すだけでクリップボードに保存されます。

### 2. 右クリックメニューからコピー
Webページ上のリンクや、ページ自体の右クリックメニューから `Title Capture...` を選択することで、情報をキャプチャできます。

### 3. 設定の変更
ポップアップ内の設定アイコン、または拡張機能のオプションページから、出力フォーマットをカスタマイズできます。

## 開発者向け情報

### 事前準備
- Node.js (v18 以上)
- npm

### セットアップ
```bash
npm install
```

### ビルド
```bash
npm run build
```
ビルドされたファイルは `dist/` ディレクトリに出力されます。

### Chromeへの読み込み
1. Chromeで `chrome://extensions` を開きます。
2. 右上の **デベロッパー モード** をオンにします。
3. **パッケージ化されていない拡張機能を読み込む** をクリックし、プロジェクト内の `dist/` ディレクトリを選択します。

## 構成
- **src/background.ts**: コンテキストメニューやバックグラウンド処理を担当。
- **src/popup.ts**: アイコンクリック時のポップアップUIを制御。
- **src/content.ts**: ページ内でのキャプチャ処理を担当。
- **src/options.ts**: 設定画面のロジック。
